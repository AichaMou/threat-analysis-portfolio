# Malicious Document Attack Analysis

## Overview
This report analyzes a cyber attack that started with a malicious document and led to full system compromise.

## Initial Access
The attacker used a malicious document to gain initial access.

## Execution
After opening the document, a second stage payload was executed.

## Network Activity
The system communicated with an external server (possible C2).

## Discovery
The attacker performed internal reconnaissance.

## Privilege Escalation
Privileges were escalated to gain higher access.

## Conclusion
This attack follows a typical multi-stage intrusion using social engineering and staged payload execution.
